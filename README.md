# 光流

## 生成光流
```python
from gen_flow import gen_flow_circle

case = 1  # 1 or 2
height = 101
width = 101
if case == 1:
    flow = gen_flow_circle([height//2, width//2], height, width)
    flow = flow / 3
else:
    flow = gen_flow_circle([0,0], height, width)
    flow = flow / 5

print(flow[:,:,0])
print(flow[:,:,1])
```
